Prinsip Induksi Sederhana

Metode pembuktian matematis untuk pernyataan yang melibatkan semua bilangan bulat positif.

Definisi

Misalkan p(n) adalah pernyataan perihal bilangan bulat positif. Kita ingin membuktikan bahwa p(n) benar untuk semua bilangan bulat positif n.

Untuk itu, cukup ditunjukkan dua hal:

(1) Basis Induksi : p(1) benar (2) Langkah Induksi: Jika p(n) benar, maka p(n+1) juga benar untuk setiap n ≥ 1 Maka → p(n) benar untuk semua n ∈ ℤ⁺
01
Basis Induksi

Buktikan p(1) benar — yaitu pernyataan berlaku untuk nilai awal n = 1. Ini adalah "domino pertama" yang kita dorong.

02
Hipotesis Induksi

Asumsikan p(n) benar untuk suatu bilangan bulat positif n tertentu. Asumsi ini disebut hipotesis induksi.

03
Langkah Induksi

Gunakan hipotesis induksi untuk membuktikan bahwa p(n+1) juga benar. Ini adalah inti dari metode induksi.

04
Kesimpulan

Bila kedua langkah (basis & induksi) telah dibuktikan, maka p(n) terbukti benar untuk semua n bilangan bulat positif.

Analogi: Efek Domino

Induksi matematika bekerja seperti barisan domino. Dorong domino pertama (basis), dan semua berikutnya akan jatuh (langkah induksi).

→

Contoh : Jumlah Bilangan Bulat

Buktikan bahwa jumlah n bilangan bulat positif pertama adalah n(n+1)/2.

Proposisi

p(n) : 1 + 2 + 3 + … + n = n(n+1) / 2 Berlaku untuk semua n ∈ ℤ⁺ (bilangan bulat positif)
i
Basis Induksi
n = 1 ▼

Tunjukkan bahwa p(1) benar:

Ruas kiri : 1 Ruas kanan : 1 · (1 + 1) / 2 = 1 · 2 / 2 = 2 / 2 = 1 ✓ Ruas kiri = Ruas kanan → p(1) terbukti benar
ii
Hipotesis Induksi
asumsi p(n) ▼

Misalkan p(n) benar untuk suatu bilangan bulat positif n, yaitu kita asumsikan:

1 + 2 + 3 + … + n = n(n+1) / 2 … (hipotesis induksi)
iii
Langkah Induksi
buktikan p(n+1) ▼

Harus diperlihatkan bahwa p(n+1) juga benar, yaitu:

1 + 2 + … + n + (n+1) = (n+1)[(n+1)+1] / 2 = (n+1)(n+2) / 2

Pembuktian:

1 + 2 + … + n + (n+1) = [1 + 2 + … + n] + (n+1) ← pengelompokan suku = n(n+1)/2 + (n+1) ← gunakan hipotesis induksi = (n+1) · [n/2 + 1] ← faktorkan (n+1) = (n+1) · [(n + 2) / 2] = (n+1)(n+2) / 2 ← sama dengan target p(n+1) ✓ p(n+1) terbukti benar
iv
Kesimpulan
Q.E.D. ▼

Karena langkah basis p(1) dan langkah induksi p(n) → p(n+1) keduanya telah dibuktikan benar, maka:

1 + 2 + 3 + … + n = n(n+1) / 2 berlaku untuk semua bilangan bulat positif n. ■
