ℹ️ APA ITU FUNGSI HASH?

Fungsi hash adalah fungsi yang memetakan sebuah key (k) ke suatu alamat dalam tabel hash menggunakan operasi modulo.


Rumus Fungsi Hash:

h(k) = k mod m

Keterangan:
  • k = key / data yang dimasukkan
  • m = ukuran tabel hash

Tujuan:

Mempercepat proses pencarian dan penyimpanan data dalam tabel hash.

📦 VISUALISASI TABEL HASH

Setiap key ditempatkan pada indeks: h(k) = k mod m

📌 KESIMPULAN

Fungsi Hash menggunakan operasi modulo untuk memetakan suatu key ke alamat tertentu dalam tabel hash. Semakin baik fungsi hash yang digunakan, semakin cepat proses pencarian dan penyimpanan data.
