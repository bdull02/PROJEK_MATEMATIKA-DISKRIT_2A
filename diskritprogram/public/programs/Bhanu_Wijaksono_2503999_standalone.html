Fungsi Boolean

Pelajari fungsi, literal, dan evaluasi nilai Boolean

Definisi Aljabar Boolean

Aljabar Boolean ditemukan oleh George Boole pada tahun 1854. Aljabar Boolean adalah sistem matematika yang bekerja dengan dua nilai saja: 0 (salah/false) dan 1 (benar/true).

Secara formal, aljabar Boolean adalah tupel <B, +, ·, ', 0, 1> yang memenuhi empat aksioma dasar:

Identitas
a + 0 = a
a · 1 = a
Komutatif
a + b = b + a
a · b = b · a
Distributif
a · (b+c) = ab + ac
a + (b·c) = (a+b)(a+c)
Komplemen
a + a' = 1
a · a' = 0
+OR (penjumlahan)
·AND (perkalian)
'NOT (komplemen)

Apa itu Fungsi Boolean?

Fungsi Boolean adalah fungsi yang peubahnya hanya bernilai 0 atau 1, dan menggunakan operator + (OR), · (AND), dan ' (NOT/komplemen).

Setiap peubah di dalam fungsi Boolean, termasuk dalam bentuk komplemennya, disebut literal.
Contoh: fungsi h(x, y, z) = xyz' terdiri dari 3 literal: x, y, dan z'

Contoh Fungsi Boolean

Klik salah satu fungsi untuk mencoba mengevaluasinya.

f(x) = x 1 peubah, 1 literal
f(x, y) = x'y + xy' + y' 2 peubah, 5 literal
f(x, y) = x'y' 2 peubah, 2 literal
f(x, y) = (x + y)' 2 peubah, 2 literal
f(x, y, z) = xyz' 3 peubah, 3 literal
