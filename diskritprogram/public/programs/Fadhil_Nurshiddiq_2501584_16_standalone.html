Σ Penjumlahan Deret

Dalam matematika diskrit, penjumlahan deretan (sumasi) adalah proses menjumlahkan suku-suku dari suatu barisan. Ini direpresentasikan menggunakan notasi sigma Σ. Berdasarkan jenisnya, penjumlahan dapat berupa deret aritmatika dengan beda tetap, deret geometri dengan rasio tetap, atau penjumlahan ganda (nested) yang sering digunakan dalam analisis algoritma.

1

Deret Aritmatika ✅

S = n/2 × (2a + (n−1)d) ▾
Rumus: Sn = n/2 × (2a + (n−1)d)
a = suku pertama  ·  d = beda  ·  n = jumlah suku
2

Deret Geometri ✅

S = a(rⁿ−1) / (r−1) ▾
Rumus: Sn = a(rn−1) / (r−1)  [r ≠ 1]
a = suku pertama  ·  r = rasio  ·  n = jumlah suku
3

Deret Bilangan Asli (Gauss) ✅

S = n(n+1) / 2 ▾
Rumus Gauss: S = n(n+1) / 2
Menjumlahkan 1 + 2 + 3 + … + n
4

Deret Kuadrat (∑ i²) ✅

S = n(n+1)(2n+1) / 6 ▾
Rumus: S = n(n+1)(2n+1) / 6
Menjumlahkan 1² + 2² + 3² + … + n²
5

Deret Kubik (∑ i³) ✅

S = [n(n+1)/2]² ▾
Rumus: S = [n(n+1)/2]²
Identitas: (∑i³) = (∑i)²  ·  Contoh: n=4 → 100 = 10²
