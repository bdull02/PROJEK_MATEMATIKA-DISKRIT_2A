Relasi Invers & Fungsi

Eksplorasi interaktif relasi, invers, dan pengujian sifat fungsi — lengkap dengan perhitungan manual tiap kasus

Definisi

Relasi

Relasi R dari himpunan A ke B adalah himpunan pasangan terurut (a,b) dimana a ∈ A dan b ∈ B.

R ⊆ A × B
Definisi

Relasi Invers

Relasi invers R⁻¹ diperoleh dengan membalik setiap pasangan terurut dalam R.

R⁻¹ = {(b,a) | (a,b) ∈ R}
Syarat

Fungsi

Relasi f adalah fungsi jika setiap elemen domain tepat dipetakan ke satu elemen kodomain.

∀a ∈ A, ∃! b ∈ B : (a,b) ∈ f
Syarat

Fungsi Bijektif

Fungsi bijektif (injektif + surjektif) menjamin invers R⁻¹ juga merupakan fungsi.

f⁻¹ ada ⟺ f bijeksi

1

Fungsi Bijektif — Invers adalah Fungsi

R = {(1,a),(2,b),(3,c)} — bijeksi sempurna, invers R⁻¹ juga fungsi

Input Relasi
Diagram Pasangan Terurut
Invers R⁻¹:
📝 Perhitungan Manual — Kasus 1
1
Tulis relasi R:
R = {(1,a), (2,b), (3,c)}
Domain A = {1,2,3}  |  Kodomain B = {a,b,c}
2
Uji: Apakah R fungsi?
• 1 → a ✓ (tepat satu)   • 2 → b ✓ (tepat satu)   • 3 → c ✓ (tepat satu)
∴ R adalah FUNGSI
3
Uji injektif (satu-satu):
f(1)=a, f(2)=b, f(3)=c — semua nilai di kodomain berbeda.
∴ R INJEKTIF
4
Uji surjektif (onto):
Range R = {a,b,c} = Kodomain B = {a,b,c} — setiap elemen terpetakan.
∴ R SURJEKTIF
5
Hitung Relasi Invers R⁻¹:
Balik setiap pasangan (a,b) → (b,a):
R⁻¹ = {(a,1), (b,2), (c,3)}
6
Apakah R⁻¹ juga fungsi?
a→1 ✓   b→2 ✓   c→3 ✓ — setiap elemen tepat satu pemetaan.
∴ R⁻¹ adalah FUNGSI (karena R bijeksi)
✅ Kesimpulan: R bijektif, sehingga R⁻¹ ada dan merupakan fungsi. Range(R⁻¹) = {1,2,3} = A.

Fungsi Injektif — Invers Bukan Fungsi Total

Input Relasi
Pasangan Terurut
Invers R⁻¹:
📝 Perhitungan Manual — Kasus 3
1
Relasi R:
R = {(1,a),(2,b),(3,c)}
Domain A = {1,2,3,4} | Kodomain B = {a,b,c,d}
2
Uji fungsi:
1→a ✓, 2→b ✓, 3→c ✓, tapi 4 ∈ A tidak punya pasangan!
∴ R adalah fungsi parsial (domain efektif {1,2,3} ≠ A)

Uji injektif:
f(1)=a, f(2)=b, f(3)=c — semua berbeda ✓
∴ INJEKTIF

∴ TIDAK SURJEKTIF
5

R⁻¹ = {(a,1),(b,2),(c,3)}
Domain R⁻¹ = {a,b,c} ≠ B (d tidak punya pasangan di R⁻¹)
6
Apakah R⁻¹ fungsi total?
a→1 ✓, b→2 ✓, c→3 ✓ — tepat satu pemetaan, tapi d tidak terpetakan.
∴ R⁻¹ fungsi parsial (bukan fungsi total dari B ke A)
⚠️ Kesimpulan: R injektif tapi tidak surjektif (bukan bijeksi), sehingga R⁻¹ ada tapi bukan fungsi total dari B ke A.

f(1)=a dan f(2)=a → dua elemen domain ke nilai yang sama!

Relasi Umum — Komposisi Relasi

R₁ = {(x,y)|y=2x}, R₂ = {(x,y)|y=x+1} pada A={1,2,3}

Input Relasi
Pasangan Terurut
R₁:
R₂:
Invers R₁⁻¹:
📝 Perhitungan Manual — Kasus 5
1
Relasi R₁ (y = 2x):
x=1: y=2 → (1,2) | x=2: y=4 → (2,4) | x=3: y=6 → (3,6)
R₁ = {(1,2),(2,4),(3,6)}
2
Relasi R₂ (y = x+1):
x=2: y=3 | x=4: y=5 | x=6: y=7
R₂ = {(2,3),(4,5),(6,7)}
3
Komposisi R₂∘R₁ (R₁ dulu, lalu R₂):
• (1,2)∈R₁ dan (2,3)∈R₂ → (1,3) ✓
• (2,4)∈R₁ dan (4,5)∈R₂ → (2,5) ✓
• (3,6)∈R₁ dan (6,7)∈R₂ → (3,7) ✓
R₂∘R₁ = {(1,3),(2,5),(3,7)}
4
Invers R₁⁻¹:
R₁⁻¹ = {(2,1),(4,2),(6,3)} — merepresentasikan x = y/2
5
Invers R₂⁻¹:
R₂⁻¹ = {(3,2),(5,4),(7,6)} — merepresentasikan x = y-1
6
Verifikasi: (R₂∘R₁)⁻¹ = R₁⁻¹∘R₂⁻¹
(R₂∘R₁)⁻¹ = {(3,1),(5,2),(7,3)}
R₁⁻¹∘R₂⁻¹: (3,2)∈R₂⁻¹ ∧ (2,1)∈R₁⁻¹ → (3,1) ✓   (5,4)∈R₂⁻¹ ∧ (4,2)∈R₁⁻¹ → (5,2) ✓
∴ Terbukti (R₂∘R₁)⁻¹ = R₁⁻¹∘R₂⁻¹ ✓
✅ Kesimpulan: R₂∘R₁ = {(1,3),(2,5),(3,7)} bijeksi. Invers komposisi = komposisi invers terbalik: (R₂∘R₁)⁻¹ = R₁⁻¹∘R₂⁻¹.
