Apa itu Aljabar Boolean?

Aljabar Boolean adalah sistem matematika yang bekerja dengan nilai benar (1 / TRUE) dan salah (0 / FALSE). Dikembangkan oleh George Boole pada abad ke-19, aljabar ini menjadi fondasi utama dalam desain rangkaian digital, pemrograman logika, dan ilmu komputer modern.

Setiap ekspresi Boolean menghasilkan salah satu dari dua nilai: 1 (TRUE) atau 0 (FALSE). Operasi dasar yang digunakan adalah AND, OR, NOT, dan XOR. Tabel kebenaran adalah alat untuk mendaftar semua kemungkinan kombinasi input dan output dari suatu operasi Boolean.

🔌

Operasi Gerbang Logika

⊓
AND
Menghasilkan 1 hanya jika semua input bernilai 1. Jika ada satu input 0, hasilnya 0.
A AND B = A · B
⊔
OR
Menghasilkan 1 jika minimal satu input bernilai 1. Hasilnya 0 hanya jika semua input 0.
A OR B = A + B
¬
NOT
Membalik nilai input. Jika input 1, output 0. Jika input 0, output 1. Disebut juga inverter.
NOT A = Ā
⊕
XOR
Menghasilkan 1 jika input berbeda nilainya. Jika input sama (keduanya 1 atau 0), hasilnya 0.
A XOR B = A ⊕ B

Contoh Soal Interaktif

Klik tombol nilai input untuk mengubahnya antara 0 dan 1, lalu tekan "Hitung" untuk melihat hasil.

1

Ekspresi AND Sederhana

Hitung nilai ekspresi: Y = A AND B
Tentukan output Y untuk berbagai kombinasi nilai A dan B!
2

Ekspresi OR Sederhana

Hitung nilai ekspresi: Y = A OR B
Kapan nilai Y bisa bernilai 0?
3

Ekspresi NOT (Inverter)

Hitung nilai ekspresi: Y = NOT A
Jika A = 1, maka Y = ? dan jika A = 0, maka Y = ?
4

Ekspresi XOR — Beda Nilai

Hitung nilai ekspresi: Y = A XOR B
Coba buktikan bahwa XOR menghasilkan 1 hanya jika nilai A dan B berbeda!
5

Ekspresi Gabungan 3 Variabel

Hitung ekspresi gabungan: Y = (A AND B) OR (NOT C)
Ini adalah contoh ekspresi Boolean kompleks dengan 3 variabel dan 2 operasi berbeda.
